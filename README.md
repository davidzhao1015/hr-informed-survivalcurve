In rare disease modeling, we often face a common challenge: **how to derive survival curves when only limited evidence is available**.

This Excel template and Python notebook demonstrate a simple but practical approach - **deriving a target survival distribution from a base survival curve and a hazard ratio**. By converting survival proportions to hazards, applying the hazard ratio, and reconstructing the survival curve, analysts can quickly generate age-specific survival and mortality estimates.

The template automates the full workflow and includes built-in visualization for quick validation, making it a **transparent and reusable tool** for epidemiologists and health-economic modelers working with limited data.
